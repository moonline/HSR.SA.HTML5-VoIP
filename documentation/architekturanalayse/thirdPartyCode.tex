\section{Third Party Code}
	Für den Unterbau und das Testing werden die folgenden Frameworks genutzt:
	\begin{itemize}
		\item{adapter.js}
		\item{AngularJS}
		\item{jQuery}
		\item{LESS}
		\item{RequireJS}
		\item{QUnit}
	\end{itemize}

	Für das Benutzerinterface werden die Iconfonts "`Batch"' und "`Ligature"'
	verwendet.

	von Google verwendet (SDPService).
